# Diabetes-prediction
Machine learning model to predict if the patient has diabetes or not, trained on diabetes data from kaggle
Data is classified using KNN classifiers from Scikit-learn
Visualizied data using seaborn 

1 indicating having diabetes 0 indicating no diabetes
